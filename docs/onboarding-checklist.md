# Developer Onboarding Checklist

- Install Node.js LTS
- Install Firebase CLI
- Clone repository
- Configure Firebase aliases in `.firebaserc`
- Add required environment variables
- Install dependencies in `functions/` and `web/`
- Start emulators and validate local auth + Firestore
- Run lint/type checks
- Verify security rules with emulator tests
- Validate JSON schemas with sample payloads
